Функции вычисления параметров геометрических фигур (C++)

Консольная программа на C++ для вычисления параметров геометрических фигур: треугольника и прямоугольника. Проект выполнен в рамках лабораторной работы по работе с Git.

Возможности:

Треугольник (`triangle.cpp`)

- Проверка существования треугольника по трём сторонам
- Вычисление площади по формуле Герона
- Проверка на равнобедренность

Прямоугольник (`rectangle.cpp`)

- Вычисление периметра прямоугольника
- Вычисление площади прямоугольника
- Вычисление длины диагонали
